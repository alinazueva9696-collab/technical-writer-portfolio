# User Guide: Export Sales Report to Excel

## Overview
This guide explains how to export a sales report to an Excel file from a sales dashboard.  
It is intended for business users who need to analyze or share sales data.
## Prerequisites
Before exporting the report, make sure that:
- You have access to the Sales Dashboard.
- You are logged into the system.
- The report contains data for the selected period.

## Step-by-Step Instructions
1. Open the Sales Dashboard.
2. Select the required reporting period.
3. Click **Export** and choose **Excel (.xlsx)** format.

## Expected Result
After completing these steps, an Excel file containing sales data is downloaded.  
The file includes all data displayed in the dashboard for the selected period.

## Common Issues

**Export button is disabled**  
Make sure that a reporting period is selected and the report contains data.

**The exported file is empty**  
Check whether sales data is available for the selected period.
