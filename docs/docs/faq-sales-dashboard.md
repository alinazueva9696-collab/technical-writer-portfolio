# FAQ: Sales Dashboard Export

## What file formats are available for export?
Sales reports can be exported in Excel (.xlsx) format.

## Why is the Export button disabled?
The Export button may be disabled if no reporting period is selected or if the report contains no data.

## Can I export data for a custom date range?
Yes, users can select a custom reporting period before exporting the report.

## Who can export sales reports?
Only users with access to the Sales Dashboard can export reports.

## What should I do if the exported file does not open?
Make sure you are using Microsoft Excel or a compatible spreadsheet application.
